Common questions

        


        

          

            What is a SteamID64?
            
A SteamID64 is the unique 17-digit identifier attached to a Steam account. Unlike a display name or custom profile URL, it does not change.

          

          

            Does this work with private profiles?
            
The SteamID64 can still often be resolved, but private profiles may hide account details such as playtime, location, profile text, and groups.

          

          

            Is a Steam API key required?
            
No Steam API key is required for this lookup. The page uses Steam’s public profile XML. A hosted version should use its own small proxy or serverless function to handle browser CORS restrictions.
